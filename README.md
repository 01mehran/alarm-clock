## Features âœ¨

- Real-time digital clock (updates every second)
- 24-hour format time display (HH:MM:SS)
- Two dropdowns (select inputs) for choosing alarm hour and minute
- Set / Cancel alarm buttons
- Visual + audio alert when the alarm time is reached
- Clean and minimal user interface
- Fully typed with TypeScript
- Responsive design (mobile friendly)

## How to Use

1. The current time is always displayed at the top
2. Choose desired **hour** and **minute** from the dropdowns
3. Click **Set Alarm**
4. When the selected time arrives â†’ alarm will trigger (sound + visual effect)
5. Click **Stop Alarm** to turn off the alarm
